# valentinesday

Landing page de San Valentín con Next.js + Tailwind.

## Estado de commits locales
Si no ves cambios en remoto, recuerda que aquí se generan **commits locales** en la rama actual.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm test`

## Nota del entorno
En este entorno, `npm install` puede fallar por políticas de red (`403`), por lo que los scripts pueden no ejecutarse hasta tener acceso al registry.
